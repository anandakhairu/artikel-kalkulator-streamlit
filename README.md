# 🐍 Building a Simple Application using Streamlit on Python

Artikel tentang cara membangun aplikasi kalkulator sederhana 
menggunakan Python dan Streamlit.

## 📝 Tentang Artikel
Artikel ini membahas langkah-langkah membuat aplikasi 
berbasis web sederhana menggunakan library Streamlit 
pada bahasa pemrograman Python, mulai dari instalasi 
hingga pembuatan kalkulator interaktif.

## 🛠️ Teknologi yang Dibahas
- Python
- Streamlit
